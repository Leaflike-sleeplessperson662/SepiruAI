## 📌 About this application

SepiruAI helps you understand your data. It uses artificial intelligence to scan CSV files, find patterns, and create charts. You upload a file and ask questions about it. The software handles the complex math and logic. It turns rows and columns into insights without requiring you to write code or learn statistics.

## 🛠️ System requirements

This application runs on Windows 10 or Windows 11. Your computer needs at least 8 gigabytes of memory to process large files. You need an active internet connection for the first time you run the tool because it downloads necessary AI model components. You do not need any special software like Python or R installed on your machine. The application contains everything it needs to function.

## 📂 Preparing your data

SepiruAI works with CSV files. These files store data in a grid format. Use Excel or Google Sheets to ensure your file follows these rules:
- The first row must contain headers. Headers identify the data in each column, such as "Date," "Sales," or "Customer Name."
- Delete any extra rows at the top of the file that contain titles or notes.
- Ensure the data remains in a single sheet.
- Save the document as a Comma Separated Values (.csv) file.

## 🚀 Running the software 🔍

Open the application by double-clicking the icon. A black window will appear for a moment, followed by a web browser screen. This screen serves as your main dashboard.

1. Click the "Upload" button.
2. Choose your CSV file from your computer.
3. Wait for the green status bar to confirm the file upload.
4. Type a question about your data in the chat box. For example, "Show me the trend of sales over time" or "What is the average value in the profit column?"
5. The AI will generate a response and create a chart for you.
6. Click the icons next to your charts if you want to export your results as images or PDF documents.

Type "Forecast the next three months" to see a projection based on your historical numbers. The software looks at past cycles and patterns to suggest what happens next. Use this for budget plans, inventory stock, or sales tracking.

## 🛠️ Troubleshooting common issues

If you encounter a problem, follow these tips to solve it:

- The app does not open: Ensure that no other pending updates are running on your Windows system. Restart your computer and try again.
- The screen remains blank: Refresh your browser window by pressing F5. 
- Large files cause slow performance: If your file contains more than 100,000 rows, split it into smaller files before uploading. The application works best with files under 50 megabytes.
- AI answers seem wrong: Check your CSV file for empty cells or misspelled headers. Clean data leads to better answers.
- Lost chart images: Look at your computer's "Downloads" folder. The app saves all exported images there automatically.
